###### 1.Function
- Function là 1 khối lệnh được đặt tên, có thể gọi lại nhiều lần mà không cần viết lại code
###### 2.Cách khai 
- cú pháp: function tenHam() {
      // các dòng lệnh bên trong
      }
- Quy tắc đặt tên hàm:
    - Dùng camelCase: tinhTong, layDuLieu,..
    - Nên bắt đầu bằng động từ: tinh..., lay....
    - Tên phải diễn tả hành động mà hàm thực hiện
    Khai báo function chưa chạy code bên trong.Code chỉ chạy khi bạn gọi hàm
###### 3.Gọi hàm- Function thực 
- Gọi hàm = tên hàm + dấu ()
- Parameter: tên biến đặt trong () khi khai báo 
- Argument(đối số):Giá trị truyền vào () khi gọi
###### 5.Return: gửi kết quả về nơi gọi
