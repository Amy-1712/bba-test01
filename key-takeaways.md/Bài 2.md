1. Comment code
     - comment 1 dòng: // đoạn code ( phím tắt: cmd / )
     - comment nhiều dòng: /* đoạn code */
   2. Khai báo biến:
     let "tên biến" = "giá trị";

     console.log(tên biến);
   3. Khai báo hằng: 
      const "tên biến" = "giá trị";
=>>> Dùng let chỉ khi chắc chắn cần gán lại giá trị, còn lại dùng const - giúp code an toàn
  4. Kiểu dữ liệu 
    - Number
     - String
     - Boolean
     Dùng typeof để lấy kiểu dữ liệu
        console.log (typeof "tên biến")
5. Toán tử so sánh
   - so sánh bằng : == hoặc ===
   - so sánh không bằng: !==
   - so sánh lớn hơn, nhỏ hơn: > < >= <=
   3. Toán tử logic
     - && (AND)
     - || (OR)
 4. Toán tử 1 ngôi
   - Prefix: toán tử nằm ở phía trc: tăng trc, trả về sau: ++x, --x
   - Postfix: toán tử nằm ở phía sau: tăng sau, trả về trc: x++, x--
